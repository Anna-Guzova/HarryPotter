# 🧙‍♂️ Harry Potter Characters App

## ✨ About the project

This is a Harry Potter themed web application built with React.

The app allows users to explore characters from the wizarding world in a visually styled interface inspired by Hogwarts.

It was created as a learning project and became one of my first React applications.

## 🚀 Features

* Browse characters from the Harry Potter universe
* View students and Hogwarts staff
* Filter characters by house (Gryffindor, Slytherin, Ravenclaw, Hufflepuff)
* See detailed information about each character
* Themed UI inspired by Hogwarts


## ⚙️ Tech stack

* React
* API (data fetching)
* CSS

## 💡 What I learned

* Building UI with React components
* Working with external APIs
* Managing data and rendering lists
* Creating themed interfaces

## 📌 Notes

This is a learning project focused on frontend development.
The main goal was to practice React and working with API data.
